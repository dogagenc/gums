# Front-End Boilerplate
Static Website Boilerplate with Gulp, Mustache, Sass

> **CSS:**

> - SASS (scss, desktop first grid, starter mixins & helpers)
> - Sourcemaps
> - Autoprefixer
> - Minifier

> **JS:**

> - Minifier
> - Sourcemaps


### Build

```sh
gulp build
```

### Development

```sh
gulp
```
